# PRISM MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Make the PRISM MVP runnable for Paper-1, PRISM w/o Memory, PRISM w/o Paradigm, and PRISM Full experiments.

**Architecture:** Extend `GuidedSolver` with explicit feature flags, then thread those flags through CLI scripts and evaluators. Keep implementation scoped to current package boundaries and verify behavior with focused tests.

**Tech Stack:** Python, pytest, Pydantic, Z3, SQLite-backed paradigm library.
