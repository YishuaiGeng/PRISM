- **行内容**（来自 §6.4）：
  1. 方向翻转（E-003/E-004，7 条）：`A = B - 1` 错写为 `A = B + 1`

  3. 间距误用（E-005，2 条）：无间距语义时用 `+2` 替代邻接 `±1`
  4. 跨约束等式污染（E-006，2 条）：位置比较错写为 `A = B`
